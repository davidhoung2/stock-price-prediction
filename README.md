Model 

 

我們使用訓練數據集記錄訓練模型，然後用它來預測下一周的收盤值（i.e.一周的下五個值由五個工作日組成） 

網絡輸入層的輸入數據形狀為 (5, 1)，表示使用時間序列的前五個值（i.e.一周的數據）作為輸入。僅考慮數據的一個屬性（即收盤值）。
